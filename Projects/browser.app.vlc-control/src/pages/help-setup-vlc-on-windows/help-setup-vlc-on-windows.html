Setup for VLC on Windows

  1. Run VLC and from the "Tools" menu, open the "Preferences" window ...

  2. Select "All" towards the bottom left of the window ...

  3. On the left select "Interface" and then "Main Interfaces" ...

  4. On the right, enable the "Web" ...

  5. On the left, select "Interface" and then "Main Interfaces" and then "Lua" ...

  6. On the right, enter "vlcremote" as the "Password" for the "Lua HTTP" configuration ...

  7. Press the "Save" button ...


  8. Restart VLC. The Windows Firewall may block network access to VLC ...


    Press the "Allow access" button ...

  9. Setup is now complete.
